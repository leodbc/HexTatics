# Roadmap Pós-MVP — Comunidade de Mapas

## Evoluções de produto
1. Ranking all-time com filtros de período.
2. Curadoria semanal com badge featured.
3. Perfil de criador e página pública.
4. Eventos/desafios temáticos.
5. Comentários moderados.
6. Tags/categorias.
7. Followers e notificações.

## Evoluções técnicas
1. OAuth (Google/GitHub) com RLS por usuário.
2. Painel admin para revisão de hidden/reported.
3. Job de recalculo de hot_score por cron.
4. Anti-fraude de curtidas avançado.
5. CDN/caching agressivo para feed.
